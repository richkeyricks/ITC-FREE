1. Download & Instalasi

Proses ini sama mudahnya dengan menginstal browser atau game.

1
Download Aplikasi
Buka website resmi kami dan klik tombol download. Open our official website and click the download button.
www.Telegramcopytrading.com/download
2
Jalankan Installer
File bernama ITC_Setup.exe akan terdownload. Klik 2x file tersebut. File named ITC_Setup.exe will be downloaded. Double click it.
JIKA MUNCUL PESAN "Windows Protected Your PC":
Ini normal untuk aplikasi baru. Klik tulisan kecil More info lalu klik tombol Run Anyway. IF "Windows Protected Your PC" APPEARS:
This is normal. Click small text More info then click Run Anyway.

3. Koneksi Telegram (PENTING!)

Bagian ini sering terlewat. Kita butuh izin resmi dari Telegram (API ID) agar robot bisa membaca sinyal.

Apa itu API ID?
Bayangkan Anda ingin robot masuk ke rumah (Akun Telegram) Anda untuk bekerja. API ID adalah "Kunci Duplikat" resmi yang Anda buat sendiri agar robot bisa masuk dengan aman. What is API ID?
Imagine letting a robot generally into your house (Telegram Account). API ID is the "Official Key" you create so the robot can enter safely.
sequenceDiagram participant User participant WebTelegram as my.telegram.org participant AppTelegram as Aplikasi HP Telegram User->>WebTelegram: Masuk (Input No HP) WebTelegram->>AppTelegram: Kirim Kode Login (Chat Service) Note right of AppTelegram: CEK DI APLIKASI CHAT!
BUKAN SMS! AppTelegram-->>User: Kode: Qk7... User->>WebTelegram: Input Kode User->>WebTelegram: Klik 'API Development Tools' WebTelegram-->>User: Diberi API_ID & HASH
1
Login ke my.telegram.org
Buka browser (Chrome/Safari) visit my.telegram.org. Masukkan No HP Anda (Pakai kode negara, cth: +62812...). Open browser visit my.telegram.org. Enter Phone Number (Use country code, e.g., +62812...).
!
Cek Kode OTP
Telegram akan mengirim kode ke APLIKASI CHAT TELEGRAM Anda. Bukan SMS! Cek chat dari "Telegram Service". Telegram sends code to your TELEGRAM CHAT APP. Not SMS! Check chat from "Telegram Service".
2
Buat API Baru
Di web, klik menu API development tools. Isi form sembarang saja (Cth: App Title: "MyCopy", ShortName: "mycopy1"). Klik Create. On web, click API development tools. Fill form randomly (E.g. Title: "MyCopy"). Click Create.
3
Salin ke Aplikasi ITC
Anda akan melihat App api_id (angka) dan App api_hash. Copy dan Paste keduanya ke menu Telegram Config di Aplikasi ITC. You will see api_id (number) and api_hash. Copy Paste both to Telegram Config menu in ITC App.

4. Koneksi MT5 & Mulai

1
Siapkan MT5 (Allow DLL)
Buka MT5. Klik menu Tools > Options > Expert Advisors. Centang [v] Allow DLL imports. Klik OK. Open MT5. Click Tools > Options > Expert Advisors. Check [v] Allow DLL imports. Click OK.
2
Login MT5 di ITC
Di Aplikasi ITC, masuk menu Akun MT5. Masukkan Login ID & Password Trading Anda. Klik UJI LOGIN. In ITC App, go to MT5 Account. Enter Login ID & Password. Click TEST LOGIN.

3
MULAI / START
Kembali ke Dashboard. Klik tombol besar MULAI COPIER. Selesai! Back to Dashboard. Click big button START COPIER. Done!

Troubleshooting (Masalah Umum)

Masalah / Issue Solusi / Solution
Gagal Login Telegram
Auth Key Error		 Pastikan tidak ada spasi saat copy paste API Hash.
Ensure no spaces when pasting API Hash.
Status MT5: Terputus
Disconnected		 Cek tombol "Algo Trading" di MT5 harus hijau (ON).
Check "Algo Trading" button in MT5 must be green (ON).
OTP Tidak Masuk
OTP not received		 Cek aplikasi Telegram di HP Anda. Kode masuk ke chat, bukan SMS.
Check Telegram App on Phone. Code is in chat, not SMS.
