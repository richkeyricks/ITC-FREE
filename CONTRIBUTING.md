# Contributing Guidelines

Thank you for your interest in the **ITC +AI Enterprise** project! 

## Source-Available Status
Please note that this project is **Proprietary and Source-Available**. This repository contains the official documentation, visual assets, and binary releases. The core source code is kept in a private repository to protect intellectual property.

## How to Contribute

Even though the core engine is private, you can contribute in the following ways:

### 1. Bug Reports & Feature Requests
If you find an issue or have an idea for a new feature, please use the [Issue Templates](https://github.com/richkeyricks/ITC/issues/new/choose). Be as descriptive as possible.

### 2. Documentation Improvements
If you find typos or areas in the `docs/` folder that need more clarity, feel free to open a Pull Request.

### 3. Community Support
Help other traders in the Discussion section or by sharing your successful setups and configurations.

## Code of Conduct
- Be respectful to other community members.
- Avoid spamming or promoting unrelated services.
- Focus on constructive feedback and technical excellence.
